# Eco-Terminal

Airport Energy Management System with YOLOv8 Computer Vision

## Project Overview
This project implements an intelligent energy management system for airports using YOLOv8 for passenger detection and occupancy monitoring.

## Technologies
- Python 3.10
- YOLOv8 (Ultralytics)
- Flask
- PostgreSQL
- Docker & Kubernetes
- GitHub Actions CI/CD

## DevOps Configuration
This repository contains DevOps configuration files:
- `.github/workflows/ci-cd.yml` - CI/CD Pipeline
- `docker-compose.yml` - Development Environment
- `Dockerfile` - Container Configuration
- `kubernetes/` - Production Deployment Manifests
